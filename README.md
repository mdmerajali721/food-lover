# Food Lovers

**Live Site:** [https://food-lover21.netlify.app/](https://food-lover21.netlify.app/)

## Features
1. **User Authentication:** Register/login using Email/Password or Google.
2. **Add Reviews:** Submit food reviews with images, rating, and comments.
3. **All Reviews Page:** Browse all reviews sorted by latest.
4. **Favorites System:** Save your favorite reviews and view them later.
5. **Search Functionality:** Search reviews by food name using MongoDB.
